## ✨ **Features**

### 🖥️ **VPS Management**
- Create/Delete VPS containers
- Start/Stop/Restart containers
- Live statistics (CPU, RAM, Disk)
- OS selection (Ubuntu, Debian)
- Port forwarding
- IPv4 management with UPI payments

### 💰 **Free VPS System**
- Earn VPS by inviting users
- Multiple plans (Bronze to Diamond)
- Automatic claim system
- Invite tracking

### 🤖 **AI Assistant**
- Groq LLaMA 3 integration
- Server management help

### 🛡️ **Admin Features**
- Full admin control
- User management
- Suspension system
- Purge protection
