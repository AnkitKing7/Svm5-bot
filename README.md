## ✨ **Features**

### 🖥️ **VPS Management (Direct LXC)**
- ✅ Create/Delete VPS containers automatically
- ✅ Start/Stop/Restart containers with buttons
- ✅ Live statistics (CPU, RAM, Disk, Uptime)
- ✅ Multiple OS options (Ubuntu, Debian, Fedora, Rocky Linux)
- ✅ Resource limits per container
- ✅ View container logs
- ✅ SSH access via tmate (temporary SSH links)

### 🔌 **Port Forwarding**
- ✅ Forward ports from container to host
- ✅ TCP/UDP support
- ✅ Quota system per user
- ✅ List and manage active forwards
- ✅ Random port allocation (20000-50000)

### 💰 **Free VPS System**
- ✅ Earn VPS by inviting users
- ✅ Multiple plans (Bronze to Diamond)
- ✅ Automatic claim system
- ✅ Invite tracking
- ✅ Boost rewards

### 🌐 **IPv4 Management**
- ✅ Purchase IPv4 via UPI
- ✅ Automatic payment verification
- ✅ Admin assignment
- ✅ View your IPv4 addresses
- ✅ ₹50 per IPv4 (configurable)

### 🤖 **AI Assistant**
- ✅ Groq LLaMA 3 integration
- ✅ Chat history per user
- ✅ Server management help
- ✅ 24/7 availability
- ✅ Reset conversation option

### 👤 **User Features**
- ✅ Account generator (username/email/password)
- ✅ View your generated accounts
- ✅ Check invite statistics
- ✅ Interactive help menu
- ✅ Bot info and uptime

### 🛡️ **Admin Features**
- ✅ Full admin panel
- ✅ User management
- ✅ Resource allocation
- ✅ Suspension system
- ✅ Purge protection
- ✅ Server statistics
- ✅ Add/Remove admins
- ✅ List all users and VPS

### 👑 **Main Admin Features**
- ✅ Maintenance mode
- ✅ Purge all unprotected VPS
- ✅ Protect VPS from purge
- ✅ Set CPU/RAM thresholds
- ✅ View all users

### 🔒 **Security**
- ✅ License key verification
- ✅ Role-based access control
- ✅ Suspension system
- ✅ Audit logs
- ✅ Firewall configuration
- ✅ Fail2ban integration
